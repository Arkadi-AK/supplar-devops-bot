# Telegram-Bot for Supplar-App Dev-Ops
http://t.me/supplar_bot

## Installation
```bash
python3.9 -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt

cat env_sample > .env  # change BOT_TOKEN value in .env file

python src/bot.py
```
